n8n AI Email Customer Assistant

n8n • AI • API Integrations

AI chatbot that handles customer inquiries (pricing, tracking, estimation) using shared product links and contextual rules.

Overview

Built an email assistant workflow in n8n that reads inbound messages, classifies intent, fetches data via APIs, and drafts human-like replies for review and send.

Key Flows

  • Intent detection (pricing, tracking, estimation) with rules + LLM fallback.
  • Fetch product/price data from source endpoint; sanitize and map fields.
  • Generate draft reply with guardrails; route to Gmail/Helpdesk for approval.
  • Log interactions to Airtable for auditing and analytics.

Stack

n8n, Webhooks, custom API nodes, Airtable (logging), Gmail/Helpdesk integration, optional LLM step, and validation functions in JavaScript.

Impact

  • Reduced manual replies for common inquiries.
  • Improved response consistency and time-to-first-response.
  • Created an auditable trail (Airtable) for QA and improvement.
